Dieses Node Programm gibt einen Gruß auf der Konsole aus.

# Installation

Es muss node.js installiert sein (http://nodejs.org)

Es muss nichts weiter installiert werden.

# Ausführen

Um das Programm auszuführen, reicht folgender Befehl:

> node index.js
